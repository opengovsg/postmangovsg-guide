# Telegram Formatting

**Bold**

```
<b>Postman</b>
```

Underline

```
<u>Postman</u>
```

_Italic_

```
<i>Postman</i>
```

Hyperlink ([Postman](https://postman.gov.sg/))

```
<a href="https://postman.gov.sg">postman</a>
```
